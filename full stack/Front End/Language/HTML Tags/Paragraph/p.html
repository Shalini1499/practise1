PARAGRAPH TAG

Definition

A paragraph is a group of sentences which starts with a new line. In Para we have spaces, line breaks. Pre element and horizontal rules.

Example:-

<p> This is a paragraph.</p>

<p>This is some text.</p>

Horizontal Rules

The hr tag in html Stands for horizontal rule. We use hr tag for separating different topics on a page and it is an empty tag , doesn’t require end tag.

SYNTAX

<hr…>

Lines Breaks

The
tag defines a line break. If you want to start a new line, you need to insert a line break. It is an empty tag, doesn’t require end tag.

SYNTAX

<br…>

Spaces

It is nothing but in html we can’t add extra lines or spaces, it will remove automatically extra lines & spaces.

Pre-element

The pre tag determines preformatted text.

The pre displayed in a fixed width font and the text preserves both spaces and line breaks. The pre tag requires a starting and end tag.

SYNTAX

<pre> subject..<./pre>
